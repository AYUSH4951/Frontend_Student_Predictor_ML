# Student Performance Predictor (Frontend)

This is a web-based frontend application for predicting student performance using Machine Learning. It provides a clean interface where users can input data and receive predictions through a connected backend API.

---

## Features

- Simple and intuitive user interface  
- Integration with Machine Learning backend API  
- Fast and responsive design  
- Clear display of prediction results  
- Light and dark theme support
